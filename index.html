Every Frame a Color

A look at the use of colors in Marvel Cinematic Universe
Every movie is represented by a color timeline, showing how colors change throughout each movie, and how colors differ from one movie to another. See if you can identify a movie's key moments from the colors!

Site Repo
Script for Creating Images

Images are made by extracting the average colors of each frame in the movie, plotting the frame color as a thin color bar, then arranging them in chronological order.

Created this as my way of celebrating all the awesomeness in the MCU. Ideas and code base for the images come from various Reddit posts. The name "Every Frame a Color" took inspirations from the YouTube channel "Every Frame a Painting".
